🎧 RX Audio-Test (Liveempfang im Browser)

Verbindet den Browser direkt mit dem IC-905 RX-Audio-Stream zur Live-Kontrolle. Der RX-Stream wird beim Klick automatisch gestartet.

⚫ Inaktiv
100%

🎙️ TX Audio-Test (Browser-Mikrofon → IC-905)

Sendet Audio vom Browser-Mikrofon (oder einer anderen lokalen Audioquelle) direkt an den IC-905 TX-Eingang. Die Audioquelle wird auf dem Gerät ausgewählt, auf dem der Browser läuft – nicht auf dem Container-Rechner.

⚫ Inaktiv

⚠️ Der Browser fordert Mikrofonberechtigung an. Auf HTTPS-Verbindungen wird dies von modernen Browsern erzwungen. Auf HTTP (localhost / 127.0.0.1) ist es meist trotzdem erlaubt.

WebSocket-API

RX (Empfang lesen): ws://…/api/audio/rx

TX (Senden einspeisen): ws://…/api/audio/tx

PCM-Audio wird als Binär-Frames über WebSocket übertragen. Format und Samplerate gemäß obiger Konfiguration.
